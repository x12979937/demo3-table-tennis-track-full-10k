# demo3 table tennis track full 10k

Strict regenerated MuJoCo dataset. Each archive contains one or more episode folders. Each episode folder includes `state_replay_validation.json`, `episode_XXXXXX.npz`, seven camera videos, seven raw RGB frame tarballs, seven perception annotation files with 2D boxes and pixel masks, professional paddle/ball mesh assets, and plain plus structured scene/object descriptions.

The validation file records whether the state sequence can be imported back into MuJoCo, trajectory error, contact penetration, ground penetration, speed sanity checks, mesh presence, text description presence, and task constraints such as paddle-ball hit and wall-hit records.
